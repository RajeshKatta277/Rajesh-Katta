Projects

  • Pest Detection

    VassarLabs

    Objective: Trained an object detection model to accurately identify 130 different pests in farm environments.

    Model Architecture: Utilized a two-stage Faster R-CNN network, leveraging its robust feature extraction and region proposal capabilities.

    Class Imbalance Handling: Addressed class imbalance through balanced sampling techniques,and area under ROC curve, ensuring fair representation of minority classes during training.

    Optimization & Training: Employed the SGD optimizer with StepLR learning rate scheduler for stable and efficient convergence.

    Results: Achieved a mean Average Precision (mAP) of 88% and an F1 score of 85%, demonstrating strong detection capabilities despite the challenging class imbalance.

  • Farm Boundary Segmentation

    VassarLabs

    Objective: Developed a deep learning pipeline for farm boundary segmentation using satellite imagery.

    Model Architecture: Implemented U-Net and U-Net++ for semantic segmentation, focusing on precise boundary delineation.

    Geospatial Data Handling: Processed large-scale AOI TIFF files with TorchGeo, utilizing RandomGeoSampler to extract 512x512 tiles for training.

    Custom Loss Function: Combined Boundary Loss + Dice Loss + Focal Loss, with 80% class weighting favoring the positive class to enhance boundary accuracy.

    Optimization & Scheduling: Used AdamW optimizer integrated with a cosine annealing learning rate scheduler for effective training convergence.

    Training Strategy: Implemented K-Fold Cross Validation for robust model evaluation, training on geospatially sampled tiles across epochs.

    Performance Metrics: Achieved 95% multiclass accuracy and a 96% F1 score, demonstrating strong model performance on test data.

    Impact: Successfully segmented farm boundaries with high precision, offering valuable insights for agricultural mapping and planning.
